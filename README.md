## 🛠️ Technical Stack & Architecture
* **Core:** React.js 18+ (Hooks, Context API)
* **Styling:** Tailwind CSS (Utility-first, Glassmorphism UI)
* **Icons:** Lucide React
* **Navigation:** React Router (Multi-page architecture)
* **State Management:** Global Cart Context for real-time updates.

---

## ✨ Key Features & Functionality

### 🛒 Advanced Cart System
* **Slide-over Sidebar:** Seamless shopping without leaving the current page.
* **Dynamic Calculations:** Automatically updates the **Subtotal (in Rs.)** and **Total Carbon Saved (in kg)** as you adjust quantities.
* **Persistence:** Mock cart logic that holds items throughout the user session.

### 🌗 Adaptive UI (Light & Dark Mode)
* **Day Mode:** A clean, "Airy" look with Emerald-600 and White backgrounds to symbolize nature.
* **Night Mode:** A "Deep Sea" aesthetic using Slate-900 to reduce eye strain and save device energy.
* **Seamless Toggle:** Integrated into the sticky navbar for instant theme switching.

### 🔒 Secure Auth Flow
* **Logic:** Implemented a strict **Sign-up → Login** validation. 
* **Persistence:** Login state is maintained across the app, unlocking the 'Profile' and 'Checkout' features only for registered users.

### ♻️ Eco-Friendly Logic & Benefits
We don't just sell tech; we rank it. Every product features a **Color-Coded Sustainability Badge**:
* **Green (80-100):** High-recycled content, solar-compatible, and low-wattage.
* **Yellow (50-79):** Standard tech with eco-conscious packaging.
* **Red (<50):** Standard electronic items with higher carbon footprints.
* **User Benefit:** Shopping here allows users to track their environmental impact, contributing to a cleaner Pakistan.
